# 🦷 OdontoCare

## Sobre o projeto

O **OdontoCare** é um sistema web desenvolvido para o gerenciamento de consultórios odontológicos, criado como projeto da disciplina de **Engenharia de Software**.

O objetivo do sistema é auxiliar na organização administrativa e clínica do consultório, proporcionando um gerenciamento eficiente de pacientes, consultas, pagamentos, materiais e equipamentos.

---

## Funcionalidades

* 🔐 Login com controle de acesso por usuários;
* 👤 Cadastro e gerenciamento de pacientes;
* 📝 Prontuário clínico digital;
* 🦷 Histórico odontológico;
* ⚠️ Registro de alergias;
* 🩻 Registro de exames;
* 📋 Registro de tratamentos anteriores;
* 📅 Agendamento e gerenciamento de consultas;
* ✅ Verificação de disponibilidade de horários;
* ❌ Cancelamento de consultas;
* 💳 Registro de pagamentos;
* 🧾 Emissão de recibos;
* 💰 Emissão de orçamentos;
* 📦 Controle de materiais;
* 🚨 Alerta de baixo estoque;
* 🏥 Controle de equipamentos;
* 📊 Geração de relatórios administrativos.
